About Climate Closet

Climate Closet is a small web application that connects real-time weather data to everyday clothing decisions. Instead of guessing what “feels like 47°F and windy” actually means, the site turns the forecast into concrete outfit guidance.

How it works

On the Outfit Planner page, you choose a location (via browser geolocation or manual coordinates). The app calls the public Open-Meteo API to retrieve hourly weather data for that spot, then pulls out the current temperature, humidity, and wind speed. A lightweight rules engine converts the temperature into a baseline outfit suggestion (e.g., heavy coat vs. light layers). You are able to save the outfit suggestion and view all saved outfits.

The project is designed to grow over time. In later iterations, a Supabase database will store more detailed wardrobe rules.

Intended users

This tool is for anyone who:

  • Checks the weather before getting dressed.
  • Wants a quick sanity check on whether their outfit makes sense for the day.
  • Enjoys thinking about clothing, comfort, and context (school, work, dates, gym).

Technology and design notes

Climate Closet is built with plain HTML, modern CSS, and JavaScript, Chart.js, simpleslider.js, Vercel, and Supabase. We took an emphasis on clear layouts and readable typography. The temperature-to-outfit mapping is intentionally simple and transparent so it can be extended or tuned in future iterations.

Planned stack: Node.js + Vercel API routes, Supabase (Postgres), chart.js and simpleslider.js JS libraries
